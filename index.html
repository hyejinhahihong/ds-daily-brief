★01Apple, 능동형 비서 평가 위한 사용자 시뮬레이션 프레임워크 Pare 공개 — 앱을 유한상태기계로 모델링Proactive Agent Research Environment: Simulating Active Users to Evaluate Proactive Assistants✍️ 블로그 · machinelearning.apple.comPare는 사용자의 필요를 미리 파악해 스스로 작업을 수행하는 능동형(proactive) 에이전트를 개발·평가하기 위한 프레임워크다.

기존 방식은 앱을 단순한 도구 호출(tool-calling) API로만 모델링해 사용자 상호작용의 상태 기반·순차적 특성을 반영하지 못했다는 문제를 지적한다. 이를 해결하기 위해 Pare는 애플리케이션을 유한 상태 기계(finite state machine)로 모델링하여 보다 현실적인 사용자 시뮬레이션을 가능하게 한다.

WHY IT MATTERS

디지털 비서가 사용자 요청을 기다리지 않고 먼저 행동하려면 실제 사용 흐름을 제대로 반영한 평가 환경이 필수적인데, 이 연구는 그런 환경 구축의 기초를 제공한다.

WHAT'S DIFFERENT

기존의 평평한 도구 호출 API 기반 앱 모델링과 달리 Pare는 앱을 유한 상태 기계로 표현해 상태 변화와 순차적 상호작용을 시뮬레이션한다.

02Apple, CLaRa 공개 — 연속 잠재공간에서 검색·생성 통합해 RAG 문서 길이 압축CLaRa: Bridging Retrieval and Generation with Continuous Latent Reasoning✍️ 블로그 · machinelearning.apple.comCLaRa는 검색(retrieval)과 생성(generation)을 하나의 연속 잠재공간에서 함께 최적화하는 RAG 프레임워크다.

문서를 임베딩 기반으로 압축한 벡터로 변환해 생성기에 입력되는 문서 길이를 줄이면서도 의미 정보를 보존한다. 이를 위해 질의응답과 문장 재구성(paraphrase) 기반의 데이터 합성 기법인 SCP를 도입해 압축 벡터가 핵심 정보를 유지하면서도 검색 가능하도록 만든다.

WHY IT MATTERS

WHAT'S DIFFERENT

02Apple, 사전학습 비주얼 인코더를 단 한 층만으로 이미지 생성 잠재공간에 적응시키는 방법 제시One Layer Is Enough: Adapting Pretrained Visual Encoders for Image Generation✍️ 블로그 · machinelearning.apple.com이 연구는 diffusion 모델이 학습 효율과 생성 품질을 위해 압축된 잠재공간에서 동작하는 문제를 다룬다.

최근에는 사전학습된 고품질 비주얼 표현을 VAE 내부나 생성 모델에 직접 결합하려는 시도가 늘고 있는데, 이해 지향적 특징과 생성에 적합한 잠재공간 사이의 근본적 불일치 때문에 어려움이 있다. 표현 인코더는 다양한 가설을 포착하는 고차원 잠재를 선호하지만 생성 모델은 이와 다른 특성을 요구한다는 점이 핵심 문제로 제시된다. 이 글은 단 하나의 레이어만으로 이러한 사전학습 표현을 생성 친화적 잠재공간으로 적응시키는 접근을 소개한다.

WHY IT MATTERS

사전학습된 표현을 재학습 없이 가볍게 재활용할 수 있다면 생성 모델 구축 비용과 복잡도를 크게 줄일 수 있어, 표현학습과 생성모델링을 연결하는 실용적 방향으로서 주목할 만하다.

WHAT'S DIFFERENT

기존에는 VAE 내부 정렬이나 생성 모델과의 직접 결합 등 복잡한 방식으로 표현을 통합했으나, 이 방법은 단일 레이어만 추가해 이해 지향적 특징과 생성용 잠재공간 간 불일치를 해결한다는 점에서 차이가 있다.

01PUe: Biased Positive-Unlabeled Learning Enhancement by Causal InferencePUe, 편향된 라벨링 하에서 PU 학습 개선 — MNIST·CIFAR-10·ADNI서 기존 PU 기법 능가📄 논문 · arxiv.org · NeurIPS 2023PU(Positive-Unlabeled) 학습은 소수의 양성 라벨과 다수의 미라벨 데이터로 이진 분류를 수행하는 문제인데, 기존 비용감응 방법들은 라벨이 완전히 무작위로 관측된다는 강한 가정에 의존한다.

실제로는 라벨 분포가 불균일해 선택편향이 흔하다는 점에 착안해, 저자들은 Bekker 등의 SAR-PU 성향점수 가중 프레임워크를 확장한 PUe를 제안한다. PUe는 정규화된 역확률가중(NIPW) 기반 PU 위험함수, 편향 라벨링 하 정규화 샘플가중 오차 분석, 정규화된 딥 성향점수 추정, 최신 비용감응 PU 기법과의 통합, 선택적으로 라벨된 음성 클래스 지원을 핵심으로 한다. MNIST, CIFAR-10, ADNI 실험에서 비균일 라벨 분포 조건 하 여러 PU 베이스라인 대비 성능 개선을 보였다.

WHY IT MATTERS

실세계 PU 문제에서 라벨 선택편향은 흔하지만 무시되기 쉬운데, 이를 명시적으로 모델링하는 접근은 의료영상(ADNI)처럼 라벨링이 불균일한 도메인에 실질적 도움이 될 수 있다.

WHAT'S DIFFERENT

라벨이 완전 무작위로 관측된다고 가정하는 기존 비용감응 PU 방법 및 Bekker 등의 SAR-PU 성향가중 프레임워크 대비, 정규화된 역확률가중과 딥 성향점수 추정을 결합해 편향된 라벨링 상황에서의 이론적·실증적 성능을 개선했다.
