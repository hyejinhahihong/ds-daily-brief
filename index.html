★01Apple, API 백엔드 없이 LLM으로 에이전트 학습용 합성 궤적 생성하는 방법 제안Environment-free Synthetic Data Generation for API-Calling Agents✍️ 블로그 · machinelearning.apple.comAPI 호출형 LLM 에이전트 학습에는 대량의 고품질 상호작용 궤적 데이터가 필요하지만, 이를 수집하려면 실제 실행 가능한 API와 사전에 채워진 백엔드 데이터베이스를 갖춘 환경이 있어야 해 확장성이 크게 제약된다.

이 문제를 해결하기 위해 저자들은 별도의 실행 환경 없이 LLM을 즉석 디지털 세계 모델로 활용하는 합성 데이터 생성 방법을 제안한다. API 명세만 주어지면 이 방법은 에이전트와 상태를 유지하는 환경 간의 상호작용을 모방하는 궤적을 생성한다.

WHY IT MATTERS

실제 백엔드와 API 인프라 구축 없이도 에이전트 학습 데이터를 만들 수 있다면, API 연동 에이전트 개발의 데이터 확보 병목이 크게 줄어들 수 있다.

WHAT'S DIFFERENT

기존 방식이 실행 가능한 API와 사전에 채워진 백엔드 데이터베이스를 갖춘 실제 환경을 필요로 했던 것과 달리, 이 방법은 환경 구현 없이 API 명세만으로 궤적을 생성한다.

★01AWS, Amazon Nova 2용 Self-Distilled Reasoning(SDR) 공개 — SFT 후 수학 성능 6%→70% 회복Exploring self-distilled reasoning for supervised fine-tuning with Amazon Nova✍️ 블로그 · aws.amazon.comCoT 추론 트레이스가 없는 데이터로 SFT를 하면 모델이 추론 능력을 잃는 '추론 억제' 문제가 발생하며, 실제로 수학 성능이 기존 70%에서 6%로 급락하는 것으로 나타났다.

이를 해결하기 위해 AWS는 별도 교사 모델 없이 베이스 Amazon Nova 2 Lite 모델 자신의 CoT를 추론 트레이스가 없는 데이터셋에 재사용하는 Self-Distilled Reasoning(SDR)을 제안한다. SDR을 적용하면 수학 성능을 거의 완전히 회복하면서 목표 작업 성능도 순수 SFT와 동등하거나 더 우수하게 유지된다. 세 개 벤치마크에서 검증한 결과, SDR은 모델 병합 대비 수학 성능이 비슷하면서(70% vs 68%) 목표 성능은 평균 6.5% 이상 향상시켰다.

WHY IT MATTERS

고품질 CoT 트레이스 확보 비용 때문에 추론 없이 SFT를 진행하는 경우가 많은데, SDR은 사람 주석이나 별도 교사 모델 없이 기존 SFT 데이터셋에 바로 적용 가능해 실무 커스터마이징 비용을 낮춘다.

WHAT'S DIFFERENT

기존에는 파괴적 망각(catastrophic forgetting)을 완화하기 위해 SFT 체크포인트를 베이스 모델에 병합하는 모델 병합 방식을 주로 썼지만, 이는 목표 성능 손실을 동반한다. SDR은 별도 후처리 보간 없이 학습 중 자기 자신의 추론 트레이스로 정규화를 적용해 모델 병합보다 목표·일반 성능 모두를 더 효과적으로 보존한다.

01Apple, 토큰 단위 길이 예측 모델 LenVM 공개 — 문장 단위 아닌 디코딩 스텝별 잔여 길이 추정Length Value Model: Scalable Value Pretraining for Token-Level Length Modeling✍️ 블로그 · machinelearning.apple.comApple 연구팀이 자기회귀 모델의 생성 길이를 토큰 단위로 예측하는 Length Value Model(LenVM)을 제안했다.

기존 방법들은 시퀀스 전체 단위의 거친 길이 모델링만 수행했지만, LenVM은 각 디코딩 스텝에서 남은 생성 길이를 추정한다. 이를 위해 길이 모델링을 가치 추정(value estimation) 문제로 재구성하고, 생성되는 각 토큰에 일정한 음의 보상을 부여하는 방식을 사용한다.

WHY IT MATTERS

생성 길이는 추론 비용과 추론 성능에 직접 영향을 미치는데, 토큰 단위로 남은 길이를 예측할 수 있으면 디코딩 과정에서 더 세밀한 제어와 효율화가 가능해진다.

WHAT'S DIFFERENT

기존 접근법이 시퀀스 레벨의 coarse-grained 길이 모델링에 머물렀던 것과 달리, LenVM은 토큰 레벨에서 잔여 길이를 가치 추정 문제로 정식화해 fine-grained하게 모델링한다.

02The Resolution of Causal Heterogeneity인과 이질성의 '진짜 하위그룹 수'는 없다 — resolution profile로 재정의하고 MineThatData 실험서 2~3개 그룹으로 요약📄 논문 · arxiv.org이 논문은 치료효과 이질성을 설명하는 하위그룹 수를 모델에 의존하지 않는 모집단 추정량인 resolution profile로 새로 정의한다.

이는 인과적 특성 분포의 함수로, 정해진 비율의 인과 이질성을 설명하는 최소 그룹 수를 나타내며 잠재계층 구조 없이도 모든 모집단에 대해 정의된다. 추론은 교차적합 베이지안 부트스트랩 사후분포와 영향함수 보정을 통해 이루어지고, 균일 조건부 Bernstein-von Mises 정리로 이 사후분포가 효율적 가우시안 극한에 수렴함을 보인다. 하위그룹 수의 불확실성은 모델 선택 문제가 아니라 연속 경로의 정수값 임계 비정칙성(threshold nonregularity)에서 비롯되며, 이런 knot 지점에서는 국소적으로 균등하게 일관된 단일값 선택자가 존재하지 않아 대신 동시 신뢰대역을 반전한 집합값 보고를 제시한다. 시뮬레이션과 MineThatData 이메일 실험 분석에서 방문 반응은 2~3개 그룹으로 요약되고 더 세밀한 구조는 노이즈 수준 이하로 나타났다.

WHY IT MATTERS

하위그룹 분석에서 흔히 보고되는 '몇 개 그룹'이라는 숫자가 사실은 잘 정의된 추정량이 아니라 모델에 좌우되는 결과였음을 지적하는 근본적 문제 제기로, 인과추론 및 개인화 마케팅·의료 분야의 하위그룹 보고 관행에 방법론적 재고를 요구한다.

WHAT'S DIFFERENT

기존의 잠재계층 모델 기반 하위그룹 수 추정 대비, 잠재 구조를 가정하지 않고도 정의되는 population-level 추정량(resolution profile)을 제안하며, 단일값 추정 대신 임계 비정칙성을 인정하는 집합값 신뢰구간 보고 방식을 도입한다.
