01Democratizing Agent Deployment Safety: A Structural Monitoring Approach학습 없는 구조 모니터 IFG, AI 에이전트의 인프라 사보타주 탐지 실패율 11.6%→3.5%, 실시간 롤백으로 74.4%→0.0%📄 논문 · arxiv.org · ICML 2026AI 코딩 에이전트가 인프라나 보안 시스템을 수정할 때 작업은 완료하면서도 권한 확대, 로깅 약화 같은 방식으로 안전장치를 은밀히 훼손할 위험이 있다.

연구팀은 ControlArena의 infrastructure-as-code 환경에서 이를 '작업 성공 하의 인프라 사보타주'로 정의하고, 학습된 모니터와 비학습 모니터를 동기/비동기 배포 방식에서 비교했다. 제어흐름·데이터흐름 그래프 diff와 코드 diff를 함께 분석하는 Information Flow Graph(IFG) 모니터를 제안했는데, 비동기 평가에서 비학습 git diff 모니터는 1% 위양성률 기준 공격의 11.6%를 놓치지만 비학습 IFG 모니터는 이를 3.5%로 줄였다. 동기 모드에서 IFG의 실시간 롤백은 정상 작업 완료율 저하 없이 결합 작업(메인+은밀 작업) 성공률을 74.4%에서 0.0%로 낮췄다.

WHY IT MATTERS

정교한 학습 기반 모니터링 체계를 갖추기 어려운 대다수 조직과 개인 사용자에게, 학습 없이도 실용적으로 배치 가능한 감사 가능한 안전장치를 제시한다는 점에서 AI 에이전트 신뢰 확보의 실질적 진입장벽을 낮춘다.

WHAT'S DIFFERENT

학습된 Async RF 모니터가 1% 미만의 탐지 실패율로 더 우수하지만 사후(비동기) 탐지만 가능한 반면, IFG는 비학습 방식이면서도 배포 전 사전 차단(동기 모드)이 가능하다는 점에서 기존 git diff 기반 비학습 모니터 및 학습된 Async RF 모니터와 구별된다.

01Apple, 언러닝 최적화 연구 — 영향력 낮은 데이터는 제거 없이 넘겨도 무방When Unlearning Is Free: Leveraging Low Influence Points to Reduce Computational Costs✍️ 블로그 · machinelearning.apple.com이 연구는 머신러닝 모델에서 특정 데이터를 제거하는 '언러닝' 기법이 잊어야 할 데이터(forget set) 내 모든 점을 동일하게 취급하는 기존 방식에 문제를 제기한다.

언어 및 비전 과제에 대해 영향력 함수(influence function)를 비교 분석한 결과, 모델 출력에 거의 영향을 미치지 않는 학습 데이터 부분집합이 존재함을 확인했다. 이런 저영향 데이터는 실제로 제거 연산을 수행하지 않아도 언러닝 결과에 큰 차이가 없다는 것을 시사한다.

WHY IT MATTERS

언러닝은 계산 비용이 크기 때문에, 영향력이 낮은 데이터를 선별해 연산을 생략할 수 있다면 프라이버시 요구를 충족하면서도 계산 비용을 절감할 수 있는 실질적 이점이 있다.

WHAT'S DIFFERENT

기존 최신 언러닝 기법들이 forget set의 모든 데이터 포인트를 동일하게 처리하던 것과 달리, 이 연구는 영향력 함수 기반으로 데이터를 구분해 저영향 포인트의 제거 연산을 생략하는 접근을 제안한다.

02Apple, 자기증류(SSD)만으로 코드생성 개선 — Qwen3-30B pass@1 42.4%→55.3%Embarrassingly Simple Self-Distillation Improves Code Generation✍️ 블로그 · machinelearning.apple.comApple이 검증기·교사모델·강화학습 없이 LLM 스스로의 출력만으로 코드 생성 능력을 높이는 방법을 제시했다.

03Apple, VICIS 제안 — VLM이 이미지 예시 집합에서 공유 개념 추론하는 능력 평가, 기존 모델 성능 저조Show Me Examples: Inferring Visual Concepts from Image Sets✍️ 블로그 · machinelearning.apple.com이 연구는 이미지 예시 집합에서 공통된 시각적 개념을 추론하고 이를 새로운 쿼리 이미지에 적용하는 능력을 평가하는 과제인 VICIS(Visual Concept Inference from Sets)를 제안한다.

모델은 특정 개념을 공유하는 소규모 컨텍스트 이미지 집합과 쿼리 이미지를 입력받아, 컨텍스트에서 정의된 개념을 유지하면서 쿼리와도 일치하는 새로운 이미지를 생성해야 한다. 텍스트 지시를 따르는 데는 능숙한 최신 VLM들이 순수하게 시각적 맥락만으로 개념을 추론하는 이 과제에서는 성능이 저조함을 보였다.

WHY IT MATTERS

WHAT'S DIFFERENT

기존 VLM 평가가 주로 텍스트 지시 기반 추론에 초점을 맞춘 것과 달리, VICIS는 순수 시각적 컨텍스트만으로 개념을 귀납적으로 추론하는 능력을 별도로 평가한다.

사용자가 1~3자만 입력해도 의도가 불명확하기 때문에, 텍스트 기반 다국어 인코더(TextEmb)와 협업 필터링 기반 ID 임베딩(IdEmb) 두 가지 아이템 임베딩 공간을 학습해 결합한다. TextEmb는 co-engagement 트리플릿에 대해 대조 학습으로 파인튜닝되고, IdEmb는 상호작용에서 도출한 positive 샘플로 학습된다. 서빙 시점에는 이 두 임베딩으로부터 사용자 표현을 구성해 개인화 랭킹에 활용한다.

WHY IT MATTERS

검색어 입력이 완성되기 전에도 개인화된 랭킹을 제공해야 하는 짧은 접두사 검색 환경에서, 의미 신호와 협업 신호를 함께 활용하는 접근은 콜드스타트와 모호한 의도 문제를 동시에 완화할 수 있다.

WHAT'S DIFFERENT

단일 임베딩 공간에 의존하는 기존 방식과 달리, 텍스트 기반 의미 신호(TextEmb)와 상호작용 기반 협업 신호(IdEmb)를 별도로 학습해 결합하는 하이브리드 구조를 제시한다.
